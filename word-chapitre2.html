Chapitre 2 : Saisir et modifier du texte

Dans Word, savoir saisir et modifier du texte de manière efficace est essentiel. Lorsque vous ouvrez un document, une zone blanche apparaît où vous pouvez taper votre texte à l’endroit où se trouve le curseur.

Pour saisir du texte, il suffit de cliquer dans la zone de saisie et de taper avec le clavier. Vous pouvez corriger en utilisant les touches Backspace (supprimer vers la gauche) et Suppr (supprimer vers la droite).

Pour sélectionner du texte, cliquez et faites glisser votre souris. Vous pouvez ensuite utiliser les commandes de modification comme couper, copier et coller.

  • Ctrl + C : Copier le texte
  • Ctrl + X : Couper le texte
  • Ctrl + V : Coller le texte
  • Ctrl + Z : Annuler une action

Modifier le texte consiste à ajouter des mots, supprimer des parties, corriger des fautes ou réorganiser les paragraphes pour améliorer la clarté du document.
