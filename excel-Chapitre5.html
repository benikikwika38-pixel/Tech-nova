Chapitre 5 : Graphiques dans Excel

Les graphiques permettent de transformer des données numériques en représentations visuelles claires que le cerveau humain interprète plus rapidement qu’un tableau brut. Les graphiques sont largement utilisés dans les rapports, les présentations et les analyses de tendances.

Excel propose différents types de graphiques : graphique en colonnes, graphique en lignes, graphique circulaire, graphique à barres, etc. Chaque type est adapté à un usage précis. Par exemple, un graphique en colonnes est utile pour comparer des catégories, tandis qu’un graphique en lignes est plus adapté pour observer une tendance dans le temps.

Pour créer un graphique, il faut d’abord sélectionner la plage de données qui inclut les titres et les valeurs. Ensuite, vous choisissez l’onglet « Insertion » puis le type de graphique souhaité. Une fois inséré, vous pouvez modifier le style, les couleurs, les titres et les axes du graphique.

L’ajout d’un graphique dynamique permet d’actualiser automatiquement les données lorsque les cellules du tableau changent. L’option « Graphique recommandé » aide aussi les débutants à choisir le bon type de représentation selon leur jeu de données.

Une bonne pratique lors de la création de graphiques est de toujours ajouter un titre clair, de légender les axes si nécessaire et d’éviter les éléments visuels inutiles qui peuvent rendre le graphique difficile à lire.
